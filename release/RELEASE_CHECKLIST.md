# Release Checklist

- [ ] Confirm `release/VERSION` is updated.
- [ ] Move items from `Unreleased` to a versioned section in `release/CHANGELOG.md`.
- [ ] Fill out `release/RELEASE_NOTES_TEMPLATE.md` and publish release notes.
- [ ] Verify README and LICENSE are up to date.
- [ ] Tag the release in version control.
- [ ] Publish release artifacts.
